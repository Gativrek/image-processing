\# Project 01 — Blood Cell Segmentation and Analysis



Automated segmentation and morphological analysis of red blood cells (RBCs)

in brightfield microscopy images, implemented as an ImageJ plugin.



\---



\## Overview



The plugin processes an RGB brightfield image through a classical image

analysis pipeline — background subtraction, Otsu thresholding, watershed

segmentation — and outputs a particle analysis table alongside a masked

color overlay of the detected cells.
